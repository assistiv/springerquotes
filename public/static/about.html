What is Springer Quotes?

Springer Quotes let's you search more than 80 thousand articles from Springer's open access journals. When you find a particularly interesting article you can create quotes from paragraphs or individual sentences. These quotes can be used to point fellow researchers to specific sections of articles or to reference these quotes within your own scientific work.

This application was developed during the Springer API Challenge in 2011 by Sebastian Spier. Highlighting of quotes based on Michael Donohoe's code for highlighting New York Times articles. Logo artwork by Pascal Bertram.

I hope you will find Springer Quotes useful! Let me know if you find anything that can be improved. You can contact me directly on Twitter.

Which technology powers Springer Quotes?

For people who are interested in this kind of questions, here all major services and technologies used:

  • heroku for hosting this site
  • CouchDB for the data storage
  • Ruby and Sinatra for the implementation and server DSL
  • bit.ly for shortening the links
  • Nokogiri + XSLT for the conversion of Springer's A++ format into (somewhat) readable HTML.
  • Emphasis JavaScript library for the text highlighting
  • MathJax for beautifying mathematic formulas
  • 'Sticky' buttons at the top of the article
  • Pure CSS speech bubbles for the nice speech bubbles you see on the /quotes page
  • jQuery as my JavaScript library of choice
